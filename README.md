# Jogo de Fazenda - BDD

Este projeto simula funcionalidades de um jogo estilo Stardew Valley, com foco em cultivo e pesca.

## Funcionalidades
- Plantio de sementes
- Pesca em lago

## Executar testes BDD
```bash
pip install behave
behave
```

## Estrutura
- `features/`: arquivos .feature e passos
- `jogo/`: implementação do jogo
